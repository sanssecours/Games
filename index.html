NR Not Recognized Controller input not recognized
NRSD Not Recognized Steam Disabled Controller input not recognized (Steam Input Disabled)
NRSE Not Recognized Steam Enabled Controller input not recognized (Steam Input Enabled)
NRU No Rumble No rumble support
